Hand Detection Mouse Control (手部偵測滑鼠控制)
---------------------------------------------------

本專案利用 Mediapipe 與 Google Hand Landmark 模型建立一套手部偵測滑鼠控制系統。
可以透過手部動作控制滑鼠鼠標，並為後續加入手勢滾輪或多指操作奠定基礎。

---------------------------------------------------

專案功能

1. 手部骨架偵測
   使用 Google hand_landmarker.task 模型進行手部關鍵點追蹤。

2. 滑鼠控制手勢
   - 手掌移動滑鼠光標
   - 食指觸碰拇指 → 左鍵點擊
   - 中指觸碰拇指 → 右鍵點擊
   - 食指+中指往上 → 滾輪持續向上
   - 食指+中指往下 → 滾輪持續向下
   - 握拳暫停控制 → 重新擺放手的位置

3. 計算手掌、手指距離
   可用於後續手勢操作，例如點擊、拖曳、滾輪等。

模型下載
---------------------------------
本專案使用 Google hand_landmarker.task 模型 (7.45MB)。由於模型由 Google 提供，建議從官方來源下載，下載後直接放置於專案資料夾根目錄即可：
hand_landmarker.task

使用方式：
1. 安裝依賴套件：
   pip install -r requirements.txt
2. 執行 main.py

3. 畫面中會顯示即時攝影機影像，移動手掌控制滑鼠光標。
   操作方式
   - 手掌移動滑鼠
   - 食指觸碰拇指 → 左鍵點擊
   - 中指觸碰拇指 → 右鍵點擊
   - 食指+中指往上 → 滾輪持續向上
   - 食指+中指往下 → 滾輪持續向下
   - 握拳暫停控制 → 重新擺放手的位置
